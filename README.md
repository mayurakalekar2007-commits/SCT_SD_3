# SCT_SD_3 - Sudoku Solver

## Task 3
A Python program that solves any valid Sudoku puzzle using backtracking algorithm.

## Features
- User inputs the puzzle manually
- Uses backtracking to solve
- Validates rows, columns and 3x3 boxes
- Displays puzzle before and after solving

## How to Run
```bash
python Task3.py
```

## No external libraries needed
